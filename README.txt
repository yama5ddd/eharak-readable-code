■プログラム実行手順

・Visual Studio 2019 で eharak-readable-code.sln を開く
・[プロジェクト]-[eharak-readable-code]のプロパティを選択する
・[デバッグ]-[開始オプション]-[コマンドライン引数]にdictionary-data.txtのフルパスを指定する
・Program.cs を開く
・Main関数の末尾にブレークポイントを設定する
・開始ボタンを押下してプログラムを実行する (自動的にビルドされる)
・ブレークポイントで停止したら、コンソールウィンドウにフォーカスを切り換えて、出力を確認する

■入力データファイル（dictionary-data.txt）について

・テキストファイル
・コミット済みのファイル名はdictionary-data.txtであるが、入力ファイル名自体は任意
・文字コードは任意
・入力データファイルの各行のデータを出力する
・入力データファイルの登録順(行数)をIDとする
・コミット済みのデータファイルの内容
